# Nexus Fund — Decentralized Crowdfunding & Milestone Escrow Platform

A full-stack decentralized application built on Ethereum Sepolia. Project creators raise ETH from backers; funds remain locked in a smart-contract escrow and are released per-milestone only after a majority vote from backers.

### Smart Contract Design

**`CrowdfundingEscrow`** — main contract, inherits `ReentrancyGuard`, `Ownable`, `Pausable`

| Feature | Implementation |
|---|---|
| Campaign creation | `createCampaign(title, desc, goal, duration)` |
| Funding | `contribute(campaignId)` — payable, ETH only |
| Campaign settlement | `settleCampaign(campaignId)` — called after deadline |
| Milestone management | `addMilestone`, `requestMilestonePayout` |
| Voting | `voteOnMilestone(campaignId, milestoneIndex, approve)` — 7-day window |
| Auto-release | `_tryFinalizeMilestone` called after each vote; releases if >50% approve |
| Refunds | `claimRefund(campaignId)` — available only on failed campaigns |
| NFT badges | `BackerBadge` (ERC-721) — auto-minted to first-time backers |

**Security protections:**
- `ReentrancyGuard` on all ETH-transferring functions
- `refundClaimed` mapping prevents double refunds
- `milestoneVoted` mapping prevents double voting
- `onlyOwner`-gated pause/unpause
- Creator-only access to milestone payout requests

## Setup Instructions

### Prerequisites
- Node.js ≥ 20
- pnpm (`npm install -g pnpm`)
- MetaMask browser extension
- Sepolia testnet ETH (get from a faucet)
- [Alchemy](https://alchemy.com) or [Infura](https://infura.io) RPC URL for Sepolia
- [Etherscan API key](https://etherscan.io/apis) for verification

### 1. Clone & Install

```bash
git clone <repo-url>
cd nexus

# Install contract deps
cd contract && npm install

# Install frontend deps
cd ../frontend && pnpm install
```

### 2. Configure Environment

```bash
# In contract/
cp .env.example .env
```

Edit `contract/.env`:
```
SEPOLIA_RPC_URL=https://eth-sepolia.g.alchemy.com/v2/YOUR_KEY
PRIVATE_KEY=your_wallet_private_key_without_0x_prefix
ETHERSCAN_API_KEY=your_etherscan_api_key
```

### 3. Run Tests

```bash
cd contract
npm test
```

Expected output: **26 passing** across Campaign, Funding, Refund, Voting, Escrow, and NFT test suites.

### 4. Deploy to Sepolia

```bash
cd contract
npx hardhat compile
npm run deploy:sepolia
```

The deploy script automatically writes the contract address and ABI to `frontend/lib/contractConfig.ts`.

### 5. Verify Contract

```bash
cd contract
npx hardhat verify --network sepolia <CONTRACT_ADDRESS>
```

### 6. Run Frontend Locally

```bash
cd frontend
pnpm dev
# Open http://localhost:3000
```

### 7. Deploy Frontend to Vercel

```bash
# From the frontend directory
npx vercel --prod
```

Or push to GitHub and connect the `frontend/` directory to a Vercel project.

## Using the App

1. **Connect MetaMask** — click "Connect Wallet" in the navbar; switch to Sepolia if prompted
2. **Browse campaigns** — home page shows all campaigns with filter tabs
3. **Create a campaign** — fill in title, description, goal (ETH), duration, and milestones
4. **Back a project** — contribute ETH on any active campaign's detail page; receive an NFT badge
5. **Vote on milestones** — when a creator requests a milestone payout, backers vote Approve/Reject
6. **Claim refund** — if a campaign fails (goal not met by deadline), contributors can claim full refunds

## Bonus: NFT Backer Badges

Every backer receives a soulbound-style ERC-721 NFT on their first contribution. Metadata is fully on-chain (base64-encoded JSON) and includes:
- Campaign name
- Amount contributed (wei)
- Contribution date (Unix timestamp)
- Campaign ID

The `BackerBadge` contract is deployed as an immutable child of `CrowdfundingEscrow`; only the escrow contract can mint badges.
